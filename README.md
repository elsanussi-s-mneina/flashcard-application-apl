
# Installing APL

https://www.gnu.org/software/apl/apl.html

Download GNU apl 1.8.

Untar the download. You should have a folder.

Go inside that folder.

Run 

$ ./configure
$ make
$ make install

# Running the program
$ apl -f hello.apl

# Exiting APL
Type The contents of the following 
")OFF" exactly without quotes and press enter.
